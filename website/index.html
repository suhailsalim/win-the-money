What it does

Net worth, banks, cards, deposits, investments, budgets, goals, income & tax — all tracked on-device.

📄

Statement import

Password-protected bank/card PDF statements (HDFC, Federal, Axis, ICICI, Scapia…) parsed on-device, including combined statements with FDs/RDs.

📧

Gmail auto-import

Read-only OAuth scan of transaction-alert emails and statement attachments.

📊

Budgets

Per-category caps (monthly / quarterly / annual / custom), period navigation (month, YTD, FYTD, FY), drill-in to transactions.

🏆

Goals backed by real assets

Back goals with real assets so progress updates itself; light XP/levels/badges.

📈

Investments, live prices

Live prices from Yahoo Finance (stocks/ETFs) and AMFI (mutual funds) — free, no API keys.

🧾

Indian tax engine

Income streams, payslip import, a real Indian slab engine (both regimes compared).

Privacy

The core selling point, made verifiable.

  • No backend, no analytics, no tracking, no third-party SDKs.
  • Network calls are limited to services you enable: Gmail (read-only), Setu AA, Yahoo/AMFI/FX quotes, and your chosen AI provider (aggregates only).
  • The repo is open source — the claims above are verifiable in the code.

This page collects nothing either: no analytics, no cookies, no third-party scripts or fonts. View source and check for yourself.

Build it yourself

Nidhi isn't on the App Store — it ships only via a free personal Apple developer team, so you build and sideload it yourself.

Requirements

  • Xcode 26+ / iOS 26 SDK
  • An iOS 26 device or simulator
  • (Optional) A Google Cloud OAuth iOS client ID for Gmail import
  • (Optional) Setu Account Aggregator credentials for AA bank sync
git clone https://github.com/suhailsalim/win-the-money.git
cd win-the-money
open WinTheMoney.xcodeproj

Or build for a simulator from the CLI — see the full xcodebuild command in the README.
